# Environment Variables

Use `.env.example` as the template. Do not commit real `.env` files.

## Vercel Frontend

- `NEXT_PUBLIC_APP_URL`
- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_PARTICLE_PROJECT_ID`
- `NEXT_PUBLIC_PARTICLE_CLIENT_KEY`
- `NEXT_PUBLIC_PARTICLE_APP_UUID`
- `NEXT_PUBLIC_MAGIC_PUBLISHABLE_KEY`
- `NEXT_PUBLIC_PAYPORT_NETWORK_MODE`
- `NEXT_PUBLIC_ARBITRUM_ONE_CHAIN_ID`
- `NEXT_PUBLIC_ARBITRUM_ONE_RPC_URL`
- `NEXT_PUBLIC_ARBITRUM_ONE_SETTLEMENT_CONTRACT_ADDRESS`
- `NEXT_PUBLIC_FINAL_PARTICLE_TRANSACTION_ID`
- `NEXT_PUBLIC_FINAL_SETTLEMENT_TX_HASH`

Only public, non-secret values belong in `NEXT_PUBLIC_*`.

## Railway Backend

- `NODE_ENV`
- `PORT`
- `DATABASE_URL`
- `FRONTEND_ORIGIN`
- `PAYPORT_API_SECRET`
- `ARBITRUM_ONE_RPC_URL`
- `SETTLEMENT_EXECUTOR_MAINNET_PRIVATE_KEY`
- `ETHERSCAN_API_KEY`
- `PARTICLE_PROJECT_ID`
- `PARTICLE_CLIENT_KEY`
- `PARTICLE_APP_UUID`

`SETTLEMENT_EXECUTOR_MAINNET_PRIVATE_KEY` must only exist on Railway or local private developer machines.

## Sepolia Rehearsal

- `ARBITRUM_SEPOLIA_RPC_URL`
- `ARBITRUM_SEPOLIA_PRIVATE_KEY`
- `NEXT_PUBLIC_ARBITRUM_SEPOLIA_RPC_URL`
- `NEXT_PUBLIC_ARBITRUM_SEPOLIA_SETTLEMENT_CONTRACT_ADDRESS`
